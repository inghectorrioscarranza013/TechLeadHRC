THE RULES

When a message (any text max 20 characters) is submitted it has to pass through the queue then the queue will process the message. This process includes waiting a random time (5 to 10 seconds) to use the API to write it to the database, if the message includes some letters it has to be discarded (b,c,d,t) at the end send a notification to the client that a message has been processed with an ID. The client should use the API to retrieve the messages stored in the database.
